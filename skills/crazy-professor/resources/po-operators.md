# PO Operators (Edward de Bono)

PO stands for "Provocation Operation" -- a deliberate destabilization of
accepted thinking patterns. The operator signals that what follows is
not a proposal but a provocation intended to move thinking sideways.

V1 uses three of de Bono's six operators. The remaining three (wishful
thinking, distortion, arising) are reserved for V2 to keep the initial
space focused.

## Reversal

Invert a normal relationship. If A usually acts on B, have B act on A.
If X usually comes before Y, have Y come before X. If something is
usually done for a reason, remove the reason and see what remains.

Example scaffold:
  "Normally [subject] [verb] [object]. What if [object] [verb] [subject]?"

## Exaggeration

Push a feature to an absurd extreme and observe what breaks. Not for
the extreme itself -- for what the extreme reveals about the system's
assumptions.

Example scaffold:
  "Normally [X] is done [modestly]. What if [X] is done [100x]?"
  "Normally [X] happens [once]. What if [X] happens [every minute]?"

## Escape

Identify a defining feature of the current situation and remove it
entirely. Not replace, remove. Then describe what remains.

Example scaffold:
  "Normally [subject] has [feature]. What if [feature] is gone?"
  "Normally [process] requires [input]. What if [input] does not exist?"

## Rules for All Three

- The provocation is not a solution. It is a movement-generator. If the
  provocation directly describes a usable solution, it is too tame.
- The provocation must be extreme enough that it makes the practitioner
  uncomfortable for a moment before they see a way to use it.
- The goal is not to be clever; the goal is to force sideways movement.
