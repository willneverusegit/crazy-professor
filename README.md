# crazy-professor

A Claude Code plugin that delivers a divergence generator for creative ideation.

Three voices — **first-principles-jester**, **labyrinth-librarian**, **systems-alchemist** — produce 10 provocations per call. Never advice. Each provocation combines a random archetype, a random provocation word, and a random PO operator (De Bono's Provocation Operation) to deliberately nudge your thinking away from the obvious.

## Use when

- You feel stuck on a problem
- The first idea feels too normal
- A brainstorm needs a destabilizer
- A plan converged too fast
- A feature needs a weirder starting point

## Install

```bash
claude plugin marketplace add willneverusegit/crazy-professor
claude plugin install crazy-professor --scope user
```

## Trigger

- "crazy professor"
- "provoke me"
- "break my thinking"
- "give me weird ideas"
- "shake this up"
- "too normal"
